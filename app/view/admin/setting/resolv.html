解析设置

目前仅支持在阿里云 alibabacloud.com 和 aliyun.cn 注册的域名。功能开启后，在虚拟机列表中，更多选项中会多一个“同步解析”选项。点击后，会以此虚拟机的名称（非备注）为关键词搜索A记录，若不存在则会创建新纪录；若存在，且解析值不是当前虚拟机地址，则会更新解析

功能开关：
创建后立即同步：
accessibility
domain
text_fields
code
timer
